# 段落
### 一般文本
```markdown
*斜体文本*          _斜体文本_

**粗体文本**            __粗体文本__

***粗斜体文本***            ___粗斜体文本___
```
### 特殊文本
```markdown
~~删除线~~

<u>下划线</u>

文本内容[^脚注]
[^脚注]: 脚注内容
```
### 分割线
```markdown
***         * * *
---         - - -
```

# 表
### 无序列表
```markdown

-           *           +
-           *           +
```
### 有序列表
```markdown

```
### 表格
```markdown
| 表头 | 左对齐 | 居中对齐 | 右对齐 |
| --- | :--- | :---: | ---: |
| 单元格 | 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 | 单元格 |
```

